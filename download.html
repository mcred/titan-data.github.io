Before using titan, you will need to have Docker Desktop installed, with the ability to run Linux containers. If you are using a Linux System, consult the Documentation to determine if your distribution is supported.

Once you have downloaded and extracted the distribution, simply run titan install. This will download the titan docker container that will then run in the background. That's all you need to get started with titan!

Where's Windows?

We have been wrestling with some last minute issues with the Windows build. Rather than delay further releases, we're going to start with MacOS and Linux. We will release Windows once it is stable.
